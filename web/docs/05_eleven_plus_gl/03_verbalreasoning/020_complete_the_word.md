# Complete the word

## **TYPE 20: Complete the word (IPS Type O)**

### What you are asked to do

You’re given **two completed pairs** showing the rule:

**word1 ( answer ) word2**
**word3 ( answer ) word4**
**word5 ( ? ) word6**

The bracket word is made in the **same way** each time.
You must find the missing bracket word for the third pair.

Example:
timed (**dim**) timer (**rim**) deals (**?**)
Rule shown: take the **inside 3 letters** (timed → **imd**? actually “dim” is made by removing the first letter t; timer → “imer” remove t gives “imer” not rim)
In IPS Type O the usual rule is: **remove the first letter** then take 3 letters to make a real word:

* timed → remove **t** = imed → pick **dim** using letters from *timed* without rearranging (drop t,e)
* timer → remove **t** = imer → pick **rim** (drop e)
* deals → **sea** (drop d,l)
  Answer: **sea**

(You don’t need to explain it in the test — just find the word that fits the same method.)

---

## How to solve (reliable method)

1. Look at the first two examples and work out what’s happening:

   * Are letters being **removed**?
   * Are letters being taken from **start/middle/end**?
   * Is it a **hidden word** inside each?
2. Confirm the same rule works for both pairs.
3. Apply the rule to the final word and pick the option that matches.

---

## Tips

* Usually it’s:

  * remove one letter from the word to make a shorter real word
  * or take a chunk (start/middle/end)
* Check spelling — small mistakes are common traps.
* If multiple answers seem possible, the **same exact rule** must fit all pairs.
