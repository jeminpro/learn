# Move a letter

## **TYPE 8: Move a letter (IPS Type J)**

### Short how-to

Move **one letter** from the **left word** into the **right word** to make **two new real words**.
✅ **No rearranging** — the remaining letters must stay in the same order.

### Steps

1. Try removing **one letter** from the left word (one at a time).
2. If what’s left is a **real word**, take the removed letter…
3. Insert it into the right word (start / middle / end).
4. If the new right word is also **real**, you’ve found the answer.

### Tip

Vowels (**a, e, i, o, u**) and common starters (**s, t, p, c**) are often the best letters to try first.
