**What is Place Value?**
Place value tells us **how much each digit in a number is worth** based on its **position**.

Think of each digit like a person sitting in a specific seat in a theatre â€” the *seat number* decides their *role*.
For example, in 3,582:

* The **2** is in the **ones** place â†’ worth **2**
* The **8** is in the **tens** place â†’ worth **80**

* The **3** is in the **thousands** place â†’ worth **3,000**

Each time you move **one place to the left**, the value becomes **10 times bigger**.
Each time you move **one place to the right**, it becomes **10 times smaller**.
