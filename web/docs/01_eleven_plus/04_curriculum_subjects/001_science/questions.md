# Science

## 📂 Structure Used

Each topic includes:

* WHY
* WHAT IF
* HOW
* COMPARE
* REAL LIFE
* DECISION

# 🚀 How to Use This (Important)

For EACH question:

* Ask for **full sentence answers**
* Then ask:

  * “Why?”
  * “Can you give another reason?”
  * “Can you give a real-life example?”

👉 This turns 150 questions into **450+ thinking opportunities**
