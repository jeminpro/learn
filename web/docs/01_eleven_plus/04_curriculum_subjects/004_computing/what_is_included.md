# 🟩 1. What is a Computer? (VERY HIGH PRIORITY)

## Concepts

* A computer:

  * processes it
  * gives **output**

---

## Vocabulary

* **input**
* **output**
* **process**

## Likely Questions

* “Information put into a computer is called?” → **Input**
* “Information produced by a computer?” → **Output**

# ⚠️ What NOT to Teach

Avoid:
❌ programming languages (Python, etc.)
❌ detailed coding syntax
❌ advanced computing concepts

Low return for FSCE.

# 🎯 What FSCE Will Actually Test

They will test:
✅ understanding of input/output
✅ recognising hardware vs software
✅ simple algorithms
✅ basic computing vocabulary
✅ online safety
