# 🚀 HOW TO USE THIS (IMPORTANT FOR 11+)

DT is NOT about building—it is about **thinking like a problem solver**:

Always push:

* “Why did you choose that material?”
* “What could go wrong?”
* “How would you improve it?”

👉 This builds **engineering-style reasoning (very FSCE-relevant)**
