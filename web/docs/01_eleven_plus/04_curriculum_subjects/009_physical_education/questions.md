# 🚀 HOW TO USE THIS (IMPORTANT FOR 11+)

PE is not about movement—it is about **reasoning in action**.

Always ask:

* “Why is that important?”
* “What happens if you don’t do it?”
* “Can you think of a real example?”

👉 This builds **decision-making under pressure (key FSCE skill)**
