# Word Number Codes

## **TYPE N: Word Number codes (IPS Type N)**

## What you are asked to do

You are given **four words** and **three number codes**.

* Each **letter = one digit**
* The **same letter always has the same number**
* The **codes are NOT in the same order as the words**
* One word has **no code given**

You must:

1. **Work out the letter–number mapping**
2. Answer questions such as:

   * *What is the code for … ?*
   * *What does this code mean?*

---

## How to solve (step-by-step)

1. **Look for repeated letters**

   * Words like **TILL, DEER, ROOF** are very helpful.
2. Match them to codes with **repeated digits** (e.g. 5332).
3. Build a key as you go:

   * T = 5, I = 3, L = 2
4. Use the key to:

   * encode new words
   * decode number strings

---

## Tips

* 🔍 Repeated letters = repeated digits
* ❌ Don’t assume word order matches code order
* ✏️ Write a small mapping table as you solve
* ⏱️ Once practised, these are **very fast marks**
