# Reading Information

### Short how to solve

1. **Underline key facts** (times, “more/less”, “longer/shorter”, totals).
2. Turn the facts into a **mini table** (who / how long / what time / how many).
3. Calculate carefully.
4. Check options A–E: only **ONE** will be definitely true.

### Tips

* “**X minutes longer than**” means **add** X.
* “**X minutes less than**” means **subtract** X.
* If everyone leaves at the same time, the one with the longer journey **arrives later**.
* Don’t pick “sounds right” — pick the one that **must** be true.
