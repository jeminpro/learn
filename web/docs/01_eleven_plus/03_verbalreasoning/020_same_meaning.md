# Same meaning

### What you are asked to do

You’re given **two pairs of words**.
You must choose **one answer word** that fits **both pairs** (same meaning / same idea).

Example:
(globe, planet) and (soil, mud)
Only **earth** fits both: Earth = planet, earth = soil. ✅

---

## How to solve (quick method)

1. Work out what each pair is pointing to (synonym, category, description).
2. Look for an option that matches **pair 1**.
3. Check the same option also matches **pair 2**.
4. If one fits only one pair, it’s wrong.

---

## Tips

* Often the answer is a **double-meaning word** (e.g., *earth*, *right*, *light*, *fair*).
* Say the word in a sentence with each pair — does it still make sense?
* If two options seem possible, one will usually be **slightly wrong** for one pair.
