# Word Connections

## **TYPE 10: Word Connections (IPS Type M)**


### What you are asked to do

You’ll see a sentence like:

**A is to ( … ) as B is to ( … )**

Pick:

* **ONE word** from the **first bracket**
* **ONE word** from the **second bracket**

So the relationship matches.

Example: Start is to (begin end today) as end is to (less finish send)
Start → **begin**, End → **finish**
Answer: **begin**, **finish**

---

## How to solve (step-by-step)

1. Work out the link between the two main words (**A** and **B**):

   * synonyms (same meaning)
   * opposites
   * part–whole (wheel → car)
   * category/member (rose → flower)
   * tool/job (brush → paint)
   * cause/effect (fire → smoke)
2. Choose the best match from the **first bracket** for word A.
3. Choose the best match from the **second bracket** for word B.
4. Re-read the full sentence to check it sounds sensible.

---

## Tips

* Don’t pick two “good” words separately — they must match the **same relationship**.
* Common patterns in GL VR:

  * synonym/synonym, opposite/opposite
  * place/thing found there (library/books)
  * person/job action (doctor/treat)
  * animal/sound (dog/bark)
  * object/material (table/wood)
* If stuck, test each option quickly by making mini-pairs.
