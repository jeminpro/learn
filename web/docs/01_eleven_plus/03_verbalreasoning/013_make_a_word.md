## 11+ GL Verbal Reasoning

# TYPE 13: Make a Word (IPS Type R)

### What you are asked to do

A word in brackets is formed by using **some letters from the word on the left** and **some letters from the word on the right**.

* The letters **can be rearranged**
* The bracket word uses **letters from both sides**
* The **same method** must be used to complete the second pair

---

### How to solve (step-by-step)

1. Look at the **completed example** on the left.
2. Work out:

   * which letters come from the **first word**
   * which letters come from the **second word**
3. Rearrange those letters to form the bracket word.
4. Apply the **same letter-taking pattern** to the second pair.
5. Check the answer is a **real word**.

---

### Tip

* Most answers are **4-letter words**
* Very common patterns:

  * 2 letters from each word
  * 3 letters from one word + 1 from the other
* Don’t assume order matters — **anagramming is allowed**
