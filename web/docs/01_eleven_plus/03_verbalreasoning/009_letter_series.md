## Answers + 1-line explanations

1. **IJ** — First letters go A,C,E,G,I (+2); second letters go B,D,F,H,J (+2).

2. **EV** — First letters go A,B,C,D,E (+1); second letters go Z,Y,X,W,V (−1).

3. **IK** — First letters A,C,E,G,I (+2); second letters C,E,G,I,K (+2).

4. **IL** — First letters A,C,E,G,I (+2); second letters D,F,H,J,L (+2).

5. **JI** — First letters B,D,F,H,J (+2); second letters A,C,E,G,I (+2).

6. **FO** — First letters A,B,C,D,E,F (+1); second letters T,S,R,Q,P,O (−1).

7. **PO** — Both letters go back 2 each time: Z,X,V,T,R,P and Y,W,U,S,Q,O.

8. **FE** — Both letters go back 2 each time: L,K,J,H,F and M,N,I,G,E.

9. **FS** — First letters go A,B,C,D,E,F (+1); second letters N,O,P,Q,R,S (+1).

10. **FK** — First letters A,B,C,D,E,F (+1); second letters F,G,H,I,J,K (+1).

11. **FJ** — First letters A,B,C,D,E,F (+1); second letters E,F,G,H,I,J (+1).

12. **FS** — First letters A,B,C,D,E,F (+1); second letters X,W,V,U,T,S (−1).

13. **FV** — First letters A,B,C,D,E,F (+1); second letters Q,R,S,T,U,V (+1).

14. **FO** — First letters A,B,C,D,E,F (+1); second letters J,K,L,M,N,O (+1).

15. **EL** — First letters A,B,C,D,E (+1); second letters P,O,N,M,L (−1).

16. **KZ** — First letters A,C,E,G,I,K (+2) while the second letter stays Z.

17. **KK** — Both letters stay equal and jump +2 each time: A,C,E,G,I,K.

18. **PQ** — Both letters jump +3: A,D,G,J,M,P and B,E,H,K,N,Q.

19. **PS** — Both letters jump +3: A,D,G,J,M,P and D,G,J,M,P,S.

20. **QP** — First letters jump +3: B,E,H,K,N,Q; second letters jump +3: A,D,G,J,M,P.

21. **WX** — Both letters go forward +2 each time: MN,OP,QR,ST,UV,WX.

22. **VK** — First letters Q,R,S,T,U,V (+1); second letters P,O,N,M,L,K (−1).

23. **FG** — It’s a sliding overlap: AB,BC,CD,DE,EF,FG (each pair shifts 1 forward).

24. **UF** — First letters go Z,Y,X,W,V,U (−1); second letters go A,B,C,D,E,F (+1).

25. **FH** — First letters go A,B,C,D,E,F (+1); second letters go C,D,E,F,G,H (+1).

26. **FS** — First letters A,B,C,D,E,F (+1); second letters G,J,M,P,S (+3).

27. **KT** — First letters A,C,E,G,I,K (+2); second letters H,J,N,P,R,T (+2).

28. **MX** — First letters A,C,E,G,J,M (+2,+2,+2,+3,+3); second letters I,L,O,R,U,X (+3 each time).

28. **KP** — First letters A,C,E,G,I,K (+2); second letters Z,X,V,T,R,P (−2).
